# How it works?

This tiny tuny game has 8 touch sensors, followed by 8 solenoids on it's frond and further followed by 8 leds. In this game you will have 30seconds and each LED glows randomly and you need to press the touch sensor behind  it to activate the solenoid and the speed increases gradually. # How to build it?

To replicate SolePlay, you need to follow the below mentioned steps:

1) Grab the CAD from the CAD folder
2) Grab the components from the BOM
3) Grab the code from the Firmware folder.
4) Wire up the stuff as per the circuit diagram.
5) Assemble the whole stuff in the 3d shell.

YAYYYY!! You finally replicated SolePlay.!!

# How to use it?

To use SolePlay..you need to press the touch sensor under the LED that randomly glows when you start playing it...You touch the sesnor and the solenoid activates...You wwill have 30 seconds and if you press the wrong sensor three times then a bomb erupts and you die.


# Specification

* Input: 8 capacitive touch sensor
* MCU: Xiao esp32 c3
* feedback: vibration fead and solonoid movement
* mechanical part: 8 solonoid (each with touch sensor)
* power management: 9 mosfet(8 for solonoid and 1 for capacitor)
* life saving feature: 2 capacitor(that will blust if you lose the game)
